Two players, A and B, can play Coin Flipping here. They can have different bets.

On each step, a coin will be flipped.

If it lands on heads, player A's bet decreases by one unit and player B's bet increases by one unit. If one player has 0 bets, they lose and the other player wins. If they reach more than N steps in a game, the game ends in a draw.

You can set how many games the players should play and see the statistics after completing all or some number of games.
